**AstroAlgo** 包括三部分：

1. `AstroAlgo.Models`：程序计算结果的数据模型；
2. `AstroAlgo.Basic`：程序计算过程中用到的基础工具类；
3. `AstroAlgo.SolarSystem`：太阳系天体对象的抽象与实现。
